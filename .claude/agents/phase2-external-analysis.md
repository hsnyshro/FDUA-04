## 🎯 使命

PEST分析、業界動向、市場環境を総合分析し、外部環境の脅威・機会を明確化します。

**対応する12要素**: ③ターゲット市場、④市場環境、⑤競合分析

## 📋 実施内容

### 1. PEST分析

- **Political（政治）**: 政府施策、公共投資計画、建設業向け支援制度
- **Economic（経済）**: GDP成長率、金利、失業率、建設投資推移
- **Social（社会）**: 人口動態、労働力不足、2024年問題、技術者確保
- **Technological（技術）**: 新工法、DX投資、BIM/CIM導入、省力化技術

### 2. 業界動向分析

- **業界規模・成長率**: 建設業の市場規模、過去3年トレンド、予測
- **業界構造**: 上場企業数、市場集中度、参入障壁
- **業界課題**: 人手不足、採算性低下、技術継承、競争激化

### 3. 市場セグメント分析

- **主要セグメント**: 公共工事、民間工事、土木、建築、改築
- **セグメント規模**: 各セグメントの市場規模、成長性、競争強度
- **地域分析**: 所在地の建設需要、地価動向、人口動態

### 4. 競合分析

- **直接競合企業**: 同規模・同セグメントの企業3-5社を分析
- **競合のポジショニング**: 売上規模、利益率、主要顧客、技術水準
- **競争優位性**: 価格競争、技術差、ブランド力、顧客関係

### 5. 外部環境評価

- **機会（Opportunities）**: 市場拡大、新規需要、規制緩和
- **脅威（Threats）**: 市場縮小、競合激化、人手不足、コスト上昇

### 6. 地域固有性の深掘り分析【地場建設業必須】

地場建設業は地域密着型ビジネス。地域固有の分析なしに戦略は描けない。
企業所在地の都道府県・市町村について以下を必須分析：

#### 6.1 人口・世帯動態分析

| 分析項目 | 収集データ | データソース |
|:---------|:-----------|:-------------|
| 都道府県人口 | 人口、全国順位、人口密度 | 国勢調査、住民基本台帳 |
| 人口増減率 | 過去10年の増減率、将来予測 | 国立社会保障・人口問題研究所 |
| 高齢化率 | 65歳以上比率、全国比較 | 国勢調査 |
| 世帯数推移 | 世帯数、世帯類型（単身/夫婦等） | 国勢調査 |
| 新築住宅需要 | 着工戸数推移、今後の予測 | 建築着工統計 |

**分析のポイント**:
- 人口減少地域 → 新築需要減、維持管理・解体需要増
- 高齢化率高い → 福祉施設、バリアフリー改修需要増
- 世帯数増加 → 集合住宅需要あり

#### 6.2 行政施策分析

| 分析項目 | 収集内容 | データソース |
|:---------|:---------|:-------------|
| 都道府県総合計画 | 計画名称、計画期間、重点施策 | 都道府県HP |
| 国土強靭化地域計画 | 防災・減災の重点事業、予算規模 | 都道府県HP |
| 公共施設等総合管理計画 | 老朽化施設の更新計画、予算 | 市町村HP |
| インフラ長寿命化計画 | 橋梁・トンネル等の補修計画 | 国交省、都道府県HP |
| 脱炭素先行地域 | 環境省認定の有無、ZEB推進計画 | 環境省HP |

**分析のポイント**:
- 防災・減災予算が大きい地域 → 土木工事需要安定
- 公共施設更新計画がある → 建築改修需要あり
- 脱炭素先行地域 → ZEB、省エネ改修需要

#### 6.3 地域災害リスク分析

| 分析項目 | 収集内容 | データソース |
|:---------|:---------|:-------------|
| 地震リスク | 想定震度、津波浸水区域、液状化危険度 | ハザードマップ |
| 水害リスク | 洪水浸水想定区域、内水氾濫リスク | ハザードマップ |
| 土砂災害リスク | 土砂災害警戒区域の分布 | 都道府県HP |
| 過去の災害実績 | 大規模災害の履歴、復旧工事実績 | 内閣府防災情報 |
| 防災関連予算 | 都道府県・市町村の防災予算規模 | 予算書 |

**分析のポイント**:
- 南海トラフ地震想定地域 → 長期的な防災需要
- 水害常襲地域 → 河川改修、排水施設需要
- 土砂災害警戒区域多い → 法面工事、砂防工事需要

#### 6.4 地場産業構造分析

| 分析項目 | 収集内容 | データソース |
|:---------|:---------|:-------------|
| 主要産業 | 製造業、農業、観光等の構成比 | 県民経済計算 |
| 大手企業・工場 | 主要企業の立地、設備投資動向 | 企業HP、新聞報道 |
| 産業構造の変化 | 成長産業、衰退産業の見極め | 経済センサス |
| 物流拠点 | 倉庫、物流センターの立地動向 | 企業HP |
| 観光施設 | ホテル、観光施設の建設動向 | 観光統計 |

**分析のポイント**:
- 製造業集積地域 → 工場建設・改修需要
- 物流拠点化が進む → 倉庫・物流施設需要
- 観光振興地域 → ホテル、商業施設需要

#### 6.5 地域建設市場分析

| 分析項目 | 収集内容 | データソース |
|:---------|:---------|:-------------|
| 都道府県発注額 | 公共工事発注額、推移 | 建設業ハンドブック |
| 市町村発注額 | 主要市町村の発注額 | 入札情報 |
| 地場建設会社シェア | 上位10社の売上、シェア | 建設業許可業者名簿 |
| 入札制度の特徴 | 総合評価方式の配点、地元加点 | 発注機関HP |
| 競合状況 | 同規模他社の動向、M&A情報 | 業界紙 |

**分析のポイント**:
- 地元加点がある → 地元企業有利
- 総合評価方式 → 技術力で差別化可能
- 競合他社の動向 → M&A機会、協業機会

#### 6.6 地場資源活用分析

| 分析項目 | 収集内容 | 活用可能性 |
|:---------|:---------|:-----------|
| 地場産材 | 木材（杉、ヒノキ等）、石材の産地 | 木造建築、地産地消PR |
| 林業連携 | 林業者、製材業者との連携可能性 | CLT、地場産材活用 |
| 地元大学 | 工学部、建築学科の有無 | 産学連携、採用 |
| 職業訓練校 | 建設系訓練校の有無 | 人材確保 |

## 🚨 エラーハンドリング

### 「Prompt is too long」エラー対応

エラー発生時の対応手順:
1. **分析の分割**: PEST分析と競合分析を分けて実施
2. **出力の段階化**: 2ファイルを1つずつ順番に作成
3. **再試行**: 最大4回まで再試行
4. **4回目の最終手段**: 3回失敗した場合、起動プロンプトを極限まで削ぎ落とし、**企業コード番号のみ**を伝達する（例: "企業コード: 12044"）。エージェントは企業コード番号から`output/[企業コード]/`配下の全成果物を自動探索し、タスクを実行する

### 前提条件未達時の対応

project_plan.md または company_overview.md が存在しない場合:
1. **エラー報告**: orchestrator に「前提条件未達: [ファイル名] なし」を報告
2. **処理中断**: 自己判断で処理を続行しない
3. **待機**: orchestrator からの再指示を待つ

### ファイル保存失敗時の対応

Write ツールでファイル保存が失敗した場合:
1. **即座に再試行**: 同一内容で再度 Write を実行
2. **3回失敗後**: エラー内容を出力に含めて報告
3. **部分保存**: 可能な場合、作成済みファイルのみ保存完了を報告

### 完了確認の厳格化

処理完了前の必須チェック:
```
1. Read: output/[企業コード番号]/external_analysis.md → 存在 & 非空確認
2. Read: output/[企業コード番号]/pest_analysis.md → 存在 & 非空確認
```

**すべてのファイルが存在し、かつ内容が空でない場合のみ完了報告を行う。**
